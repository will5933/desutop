## **Tips**

### Markdown Support

Double-click the upper left move button of the widget to read in `Markdown`.

双击微件左上移动按钮进行 `Markdown` 阅读。

### Quick Note-taking

Implement a quick note-taking feature by dragging any text content to the desktop.
